# 🚫 Forbidden Elements — Art Bible
## 禁止要素

> สิ่งที่ "ต้องไม่ปรากฏ" ในเกม — ไม่ว่ากรณีใดๆ
> ต่างจาก negative_prompts.md (แนวทาง) — อันนี้คือ "กฎเหล็ก"

---

## ⚔️ Violence & Weapons (ห้ามเด็ดขาด)

### ❌ ห้ามทั้งหมด
- อาวุธใดๆ (sword, gun, knife, bow, hammer, axe)
- อาวุธปืน (pistol, rifle, shotgun)
- อาวุธเวทมนตร์ (staff, wand)
- เลือด / บาดแผล / ความรุนแรง
- การสู้รบ / การต่อสู้ / combat
- การฆ่า / การตาย
- อุบัติเหตุรุนแรง
- สงคราม / การทหาร / ทหาร
- ระเบิด / ลูกไฟ / ไฟไหม้
- สัตว์ดุร้าย / สัตว์กัด

### ❌ ตัวละครต้องไม่
- ถืออาวุธ
- ทำร้ายผู้อื่น
- มีบาดแผล
- แสดงความเกลียดชัง
- มีอดีตทำร้ายผู้อื่น

### ✅ ทำได้
- ไม้บรรทัด (เรียน)
- ปากกา (เรียน)
- ร่ม (กันฝน)
- กระเป๋า
- แว่นตา
- จักรยาน (ขี่)

## 👻 Horror & Supernatural (ห้ามเด็ดขาด)

### ❌ ห้ามทั้งหมด
- ผี / ปีศาจ / สัตว์ประหลาด
- ความตาย / การสูญเสีย
- ความน่ากลัว / jump scare
- ภาพหลอน / hallucination
- พลังวิเศษ / เวทมนตร์ / magic
- การทำนาย / โชคลาง
- คำสาป / curse
- สัตว์ประหลาดหน้าตาแปลก
- ตุ๊กตาน่ากลัว

### ❌ ตัวละครต้องไม่
- มีพลังพิเศษ
- พูดคุยกับผี
- มีอดีตอันเจ็บปวด
- มี trauma
- มีปมในใจที่รุนแรง

### ✅ ทำได้
- ศาลเจ้า (เชิงสถานที่ ไม่ใช่ทางศาสนา)
- เทพ (ในเชิงวัฒนธรรม)
- นิทานพื้นบ้าน
- เรื่องเล่าของคนแก่
- ฝัน (แต่ต้องเป็นฝันดี)

## 💋 Adult Content (ห้ามเด็ดขาด)

### ❌ ห้ามทั้งหมด
- เนื้อหาทางเพศ / อือโรติก
- การเปลื้องผ้า
- ท่าทางไม่เหมาะสม
- การแต่งกายล่อแหลม
- กอดจูบ (มากกว่าการทักทาย)
- การนอนด้วยกัน
- สื่อลามกอนาจาร

### ❌ ตัวละครต้องแต่งกาย
- ชุดนักเรียนมาตรฐาน (กระโปรงยาวครึ่งเข่า)
- เสื้อคลุมถ้าหน้าหนาว
- ไม่มีเสื้อผ้าที่โชว์มากเกินไป
- ไม่มี deep neckline
- ไม่มี crop top (ในชีวิตประจำวัน)
- ชุดว่ายน้ำเต็มตัวเท่านั้น (โรงเรียน)

### ✅ ทำได้
- การทักทาย (โค้ง, จับมือ)
- การคุยกัน (subtle)
- การชอบใคร (เบาๆ ไม่ explicit)
- การอยู่ด้วยกัน (ปกติ)

## 🏛️ Political & Religious (ห้ามเด็ดขาด)

### ❌ ห้าม
- การเมือง / พรรคการเมือง
- การเลือกตั้ง
- การประท้วง
- ศาสนาเปรียบเทียบ / ดูถูก
- ความเชื่อส่วนบุคคลที่อ่อนไหว
- ประวัติศาสตร์ที่เป็นประเด็น
- การเหยียดเชื้อชาติ / เพศ / ศาสนา
- ความขัดแย้งระหว่างประเทศ
- การกระทำผิดกฎหมาย (รุนแรง)

### ✅ ทำได้
- ศาลเจ้า (ตามธรรมชาติ ไม่ใช่ทางศาสนา)
- เทศกาล (matsuri)
- ประเพณี (tea ceremony, hanami)
- การไหว้ (subtle, ไม่บังคับ)
- วัฒนธรรม (เชิงบวก)
- โดยไม่มีการตัดสิน
- ไม่สอนศาสนา

## 🌍 Social Issues (ห้าม)

### ❌ ห้าม
- ความยากจน (สุดขั้ว)
- ความรุนแรงในครอบครัว
- การกลั่นแกล้ง (bullying) แบบรุนแรง
- การเลือกปฏิบัติ
- การเสพสารเสพติด
- การฆ่าตัวตาย / depression
- ความรุนแรงทางเพศ
- การค้ามนุษย์
- การทารุณกรรมสัตว์
- ความอดอยาก
- โรคระบาด

### ⚠️ ข้อยกเว้น (ต้องจัดการด้วยความระมัดระวัง)
- การกลั่นแกล้งเล็กน้อย (subtle, มีการแก้ไข)
- ความเหงา (ต้องมีทางออก)
- ความผิดหวัง (ต้องฟื้นตัว)
- การสูญเสียเล็กน้อย (เช่น ของหาย)

### ✅ ทำได้
- ชีวิตประจำวันของคนธรรมดา
- ความสุขเล็กๆ
- การช่วยเหลือเล็กๆ น้อยๆ
- มิตรภาพ
- ครอบครัวที่อบอุ่น
- ชุมชนที่เข้มแข็ง
- การเรียนรู้
- การเติบโต

## 🍔 Brand & Trademark (ห้าม)

### ❌ ห้าม
- ใช้ชื่อ brand จริง (Nike, Apple, Coca-Cola, McDonald's)
- Logo จริง
- Trademarked character (Hello Kitty, Mario, Pikachu, Doraemon, Totoro)
- Real company name
- Real celebrity name
- Real location name (ใช้นามสมมติ)
- Real song title (ใช้ placeholder)

### ✅ ทำได้
- ใช้ชื่อทั่วไป (Ramen shop, Coffee shop, Bookstore)
- Brand สมมติ
- Logo ที่ออกแบบเอง
- ใช้บรรยากาศ ไม่ใช่ชื่อ
- Real food name (ถ้าเป็น generic เช่น ramen, onigiri)

### 💡 วิธี Handle
| Item | ❌ Wrong | ✅ Right |
|------|---------|---------|
| ร้านอาหาร | "McDonald's" | "Hamburger Shop" |
| ของเล่น | "Nintendo Switch" | "Portable Game" |
| ตู้ขายของ | "Coca-Cola" | "Cola" |
| หนังสือ | "Harry Potter" | "Magic School Novel" |
| เพลง | "BTS - Dynamite" | "Pop Song" |
| แบรนด์เสื้อ | "Nike" | "Sneaker Brand" |

## 🎨 Style Mismatches (ห้ามผสม)

### ❌ ห้ามผสม style
- Cel-shaded ในฉาก PBR
- Photoreal ในฉาก stylized
- 2D anime character ใน 3D world
- Real photograph ในฉาก 3D
- Logo / icon แบบ flat ในฉาก 3D
- Different render style ใน scene เดียวกัน

### ✅ Style เดียวเท่านั้น
- ทุกอย่างในเกม = PBR stylized
- UI = flat design แต่ consistent
- Texture = procedural หรือ photoreal (ไม่ผสม)
- Lighting = 3-point เดียวกันทั้งเกม

## 📱 Interface Forbidden (UX)

### ❌ ห้าม
- "Mute" button ที่ดูน่ากลัว
- Color = red = "danger" เป็น default
- Notification ที่ดู urgent
- Error message ที่ aggressive
- Tutorial pop-up ที่บังคับ
- Dark pattern (UX ที่หลอกผู้ใช้)
- Spam notification
- Forced ads

### ✅ ใช้
- Soft, friendly notification
- Subtle highlight
- Gentle prompt
- Encouraging message
- Optional tutorial
- Clear language
- "คุณทำได้" แทน "คุณทำผิด"
