01 — THE CONCEPT

A Day in Setagaya

世田谷での一日

Modern Tokyo, late spring. A second-year high school student cycles through 5 zones in one ordinary day — and finds that ordinary is beautiful.

What it is

เกม walking sim ที่ไม่มี win condition ไม่มี mission ไม่มี combat — แค่ผู้เล่นเดินสำรวจ 5 โซน คุยกับ NPC ซื้อของกิน นั่งอ่านหนังสือ และฟังเสียงชีวิตจริง

Visual style: Ghibli-pastel modern — soft edges, warm lighting, สีพาสเทล ไม่ glossy ไม่ cyberpunk ไม่ photorealistic

ผู้เล่นเลือกได้ 2 เพศ เปลี่ยนชุดได้ ใช้จักรยานได้ ถ่ายรูปในใจได้ แต่ไม่มี HP ไม่มี EXP ไม่มี fail state

TH
อธิบาย — narrative ใช้ใน dialog, hint, story
EN
Label — action HUD, menu, buttons
JA
Authentic — signage ป้ายร้าน, ชื่อเฉพาะ

04 — DESIGN PRINCIPLES

10 rules of the world

設計原則

ทุก asset / scene / code ต้องเช็คกับ 10 ข้อนี้ก่อน ship

01

🌡️ Mood > Detail

เลือก "ฉากเรียบง่าย แต่อบอุ่น" เหนือ "ฉากละเอียด แต่ดูแข็ง"

02

🌅 Time Tells Story

07:00 = แสงเช้า · 12:00 = แสงแข็ง · 17:30 = ทอง · 19:00 = ค่ำ · 22:00 = ดาว

03

🤫 Silence is OK

เกม "เงียบ" ได้ — เงียบแล้วรู้สึก "อยู่คนเดียว" ตามสไตล์ญี่ปุ่น

04

🌐 Bilingual = Respect

TH = อธิบาย · EN = label · JA = authentic ไม่แปลตรงตัว

05

👤 Character > Object

ทุก object มีเจ้าของ — รองเท้าเรียง แปรงสีฟันใช้แล้ว หนังสือเปิดค้าง

06

🚶 Walk, Don't Run

เกมนี้คือการ "ใส่ใจ" ไม่ใช่ "เร่งรีบ" — walking pace เป็น default

07

🔄 Loop is OK

ทำกิจวัตรซ้ำได้ แต่ละ loop มีรายละเอียดเปลี่ยน (คน แสง เพลง)

08

🎵 Sound = Half The Game

BGM เป็น background ที่รู้สึกได้ แต่ไม่ตั้งใจฟัง — ambient สำคัญกว่า BGM

09

🚫 No Hand-Holding

ไม่มี tutorial pop-up ไม่มี marker ให้ผู้เล่นเดินเอง หาเอง

10

🏠 Honest Mundane

"ความสวยงามของวันธรรมดา" — Wabi-sabi ไม่ใช่แฟนตาซี

05 — ANTI-REFERENCES

"ดูเหมือนใช่ แต่ผิด"

避けたいもの

สิ่งที่ห้ามทำ แม้จะดูเหมือน Tokyo — 15 ข้อ

Cyberpunk Neon Tokyo
Akihabara ยามค่ำ ไฟ neon ทุกหน้าต่าง
Anime 2D Look
Cel-shaded 3D ดูเหมือน 2D anime
Tourist Spot
Sensōji, Shibuya crossing, Tokyo Tower
Over-Saturated
"ญี่ปุ่น = สีจัด" ไม่จริง ใช้ pastel
Kawaii Mascot
Hello Kitty style ไม่ใช่ตัวละคร high school
Cold/Blue Tone
เกมเรา warm 70% + cool 30% ไม่ใช่ blue
Cosplay Uniform
ชุดนักเรียนอนิเมะ โชว์ สั้น
Rain Always
Tokyo ไม่ได้ฝนตกตลอด เน้น spring clear
Dense Crowd
Shibuya crossing 300+ คน ไม่ใช่ residential
Pet Mascot
หมา/แมวคู่ใจ เดินตาม player ไม่มี
Magic/Supernatural
ผี เทพ คาถา ไม่มี เกม realistic
Combat/Quest
mission, boss, weapon, HP ไม่มี
Hyper-Active NPC
ญี่ปุ่น = polite ไม่ hurrying ไม่ตะโกน
Cliche SFX
coin sound, level up sound ไม่มี
Generic UI
RPG bar หนาๆ เกมเรา minimal soft
