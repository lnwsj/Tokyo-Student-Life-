## Zone 1: 🏠 HOME (บ้าน)

### Hero Feature
**อพาร์ตเมนต์ 1DK** (1 room + Dining + Kitchen) + ระเบียงเล็ก

### Mood & Atmosphere
- **อบอุ่น** เหมือนถ้วยกาแฟเช้า
- **ส่วนตัว** — เป็นพื้นที่ของผู้เล่น
- **เงียบ** — ได้ยินนก + รถบนถนนไกลๆ

### Visual Style
- แสงเช้า: golden hour ผ่านม่าน
- สี: cream + wood + sakura accent
- Props: ฟูก, โต๊ะเรียน, ครัวเล็ก, ระเบียง

### Time of Day
- **07:00 - 09:00** (เริ่มเกมที่นี่)
- เปลี่ยนแสงตามเวลา — แสงเช้า → แสงเที่ยง

### Key Props
| Object | Detail |
|--------|--------|
| 🛏️ ฟูก | พับเก็บตอนเช้า ปูตอนกลางคืน |
| 🪑 โต๊ะเรียน | มีหนังสือ + สมุดโน้ต + ปากกา |
| 🍳 ครัว | หม้อหุงข้าว + ไมโครเวฟ + ตู้เย็น |
| 👕 ชุดนักเรียน | แขวนไว้ (เปลี่ยนชุด) |
| 🪟 ระเบียง | มีกระถางต้นไม้เล็ก |
| 📺 ทีวีเล็ก | เก่าๆ ไม่ค่อยได้ดู |
| 🚲 จักรยาน | จอดที่ระเบียง/ลานจอด |

### Characters (NPC)
- 👩 **แม่** — เดินอยู่ในครัว เตรียมข้าวเช้า
- (ไม่มี NPC อื่น)

### Sound Palette
- เสียงนกเช้า
- เสียงหม้อข้าวเดือด
- เสียง wind chimes (ถ้ามี)

### Color Emphasis
- **#F5F0E6** (cream) ผนัง
- **#C8A26A** (light wood) พื้น
- **#E8B4BC** (sakura) accent

## Zone 2: 🏫 SCHOOL (โรงเรียน)

### Hero Feature
**อาคารโรงเรียน 3 ชั้น** + สนาม + ต้นซากุระ

### Mood & Atmosphere
- **สดใส** + วัยรุ่น
- **คึกคัก** (ช่วงเช้า/เที่ยง)
- **อบอุ่น** — แสงแดดส่องสนาม

### Visual Style
- แสงเที่ยง: bright + soft shadow
- สี: white + navy + light green
- Props: โต๊ะเรียน, กระดาน, ตู้ล็อกเกอร์

### Time of Day
- **08:00 - 16:00** (เรียน)
- เริ่ม class 8:30, lunch 12:00, เลิก 15:30

### Key Props (ในอาคาร)
| Floor | Room | Detail |
|-------|------|--------|
| 1 | โถงทางเข้า | ตู้รองเท้า + ป้ายประกาศ |
| 1 | ห้องเรียน 1-A | โต๊ะ + กระดาน + หน้าต่างใหญ่ |
| 2 | ห้องเรียน 2-A | คล้ายชั้น 1 + ห้องสมุดเล็ก |
| 3 | ห้องชมรม | โต๊ะยาว + เก้าอี้ + แก้วชา |
| 3 | ดาดฟ้า | ราวสูง + วิวเห็น Tokyo |
| นอก | สนาม | ต้นซากุระ + ม้านั่ง + สนามบาส |

### Key Props (สนาม)
- 🌸 ต้นซากุระ 3 ต้น (กลางสนาม)
- 🏀 สนามบาส
- 🪑 ม้านั่ง 4-6 ตัว
- ⛲ น้ำพุเล็ก (ถ้ามี)

### Characters (NPC)
- 👨‍🏫 **ครู** (1 ตัว) — เดินในอาคาร
- 👫 **นักเรียน** (5-10 ตัว) — เดิน คุยกัน บางคนอ่านหนังสือ

### Sound Palette
- เสียงระฆังโรงเรียน
- เสียงนักเรียนคุยกัน (distant)
- เสียงนก + ใบไม้
- เสียง BGM เรียน ๆ (เบาๆ)

### Color Emphasis
- **#FFFFFF** (white) ผนัง
- **#2D4A6B** (navy) ประตู/หน้าต่าง
- **#C8E6A0** (light green) สนาม
- **#E8B4BC** (sakura) ต้นไม้

## Zone 3: 🚉 STATION (สถานีรถไฟ)

### Hero Feature
**สถานี JR ขนาดเล็ก** + ทางม้าลาย + ร้านสะดวกซื้อ

### Mood & Atmosphere
- **เปลี่ยนผ่าน** — ไม่ใช่บ้าน ไม่ใช่โรงเรียน
- **คนเดินผ่าน** — มี stranger ปะปน
- **เย็น** — แสงเย็นเริ่มลง

### Visual Style
- แสงเย็น 17:30: golden warm
- สี: gray + cream + orange (ไฟรถเมล์)
- Props: รถไฟ, ตู้ขายของ, ป้าย

### Time of Day
- **17:00 - 18:30** (เลิกเรียน/คนกลับบ้าน)
- เริ่มเห็นคนเยอะขึ้น

### Key Props
| Object | Detail |
|--------|--------|
| 🚉 สถานี | ticket gate, ป้าย JR, ตารางรถไฟ |
| 🚃 รถไฟ | ผ่านเป็นช่วงๆ (animation) |
| 🚦 ทางม้าลาย | 4 เลน + ไฟ |
| 🏪 ร้านสะดวกซื้อ | 7-Eleven / FamilyMart / Lawson |
| 🥤 ตู้ขายของ | 3-4 ตู้ ติดกัน |
| 🪧 ป้ายรถเมล์ | มีตารางเวลา |
| 📰 ตู้จดหมาย | สีแดงแบบญี่ปุ่น |
| 🗑️ ถังขยะแยกประเภท | 5 ใบ (ขวด/กระป๋อง/กระดาษ/พลาสติก/เผา) |

### Characters (NPC)
- 👔 **คนทำงาน** (2-3 ตัว) — เดินเร็ว
- 🧑 **นักเรียน** (3-4 ตัว) — กลับบ้าน
- 👵 **ยาย** (1 ตัว) — เดินช้า ถือของ

### Sound Palette
- เสียงรถไฟ (distant)
- เสียงประกาศสถานี (distant, garbled)
- เสียงคนคุย (distant)
- เสียงรถผ่าน
- BGM lo-fi / chill

### Color Emphasis
- **#7A7B7A** (gray) คอนกรีต
- **#FF6B35** (orange) ป้าย + ไฟ
- **#FFFFFF** (white) เส้นทางม้าลาย
- **#FFD700** (yellow) ไฟทางม้าลาย

- Props: ป้ายร้าน, ตู้ไฟ, คนเดิน

### Time of Day
- **18:30 - 22:00** (หลังเลิกเรียน/เย็น)
- **เป็น zone เดียวที่มี neon** (subtle)

### Key Props
| Object | Detail |

| 🏢 ตึกช้อปปิ้ง | 3-4 ชั้น มีร้านเล็กๆ |

| ☕ ร้านกาแฟ | เล็ก มีโต๊ะ 4 ตัว |
| 🏪 ร้านสะดวกซื้อ | เหมือน zone 3 |
| 🥖 เบเกอรี่ | มีขนมปังโชว์ |

- 🧑 **คนเดินช้อป** (15-20 ตัว) — หลากหลาย
- 👫 **คู่รัก** (2-3 คู่)

### Sound Palette

- เสียงคนคุยกัน (denser)
- เสียงจานชาม (ร้านอาหาร)
- เสียงรถผ่าน
- BGM city pop style

### Color Emphasis

## Zone 5: ⛩️ SHRINE (ศาลเจ้า + สวน)

### Hero Feature
**โทริอิสีแดง** + ศาลเจ้า + สวนญี่ปุ่น + บ่อน้ำ + สะพาน

### Mood & Atmosphere
- **สงบ** — เงียบที่สุดในเกม
- **อบอุ่น** — ไม้ + ใบไม้
- **เก่า** — มีร่องรอยเวลา

### Visual Style
- แสงเย็น 16:00: golden + soft shadow
- สี: wood brown + torii red + green
- Props: โทริอิ, ศาล, บ่อ, สะพาน, ginkgo

### Time of Day
- **16:00 - 18:00** (ก่อนค่ำ)
- เห็น sakura petals ปลิว (effect)

### Key Props
| Object | Detail |
|--------|--------|
| ⛩️ โทริอิ | ใหญ่ สีแดง vermillion 3 ประตู |
| 🏯 ศาลเจ้า (honden) | ไม้ + หลังคาสีเข้ม |
| 🎴 ศาลารอง (haiden) | ที่นักเรียนมาขอพร |
| 🔔 ระฆัง | กระดิ่งใหญ่ + เชือก |
| 🌳 ต้นซากุระ | 4-5 ต้น (กลีบร่วง) |
| 🍃 ต้นกิงโกะ | 2-3 ต้น (ใบแผ่น) |
| 🌉 สะพานไม้ | ข้ามบ่อ |
| 💧 บ่อน้ำ | มีปลา koi (red/orange) |
| 🪨 ก้อนหิน | จัดในสวนแบบ zen |
| 🏮 โคมไฟหิน | 5-6 ดวง ริมทาง |
| 🪧 แผ่นไม้ (ema) | แขวน มีข้อความ |

### Characters (NPC)
- 👵 **ยาย** (1 ตัว) — นั่งบนม้า
- 🧑 **นักเรียน** (0-2 ตัว) — มาขอพรบางที

### Sound Palette
- เสียงระฆัง (occasional)
- เสียงน้ำไหล
- เสียงนก
- เสียงใบไม้
- BGM shamisen + flute (japanese traditional)

### Color Emphasis
- **#CC4A2C** (vermillion red) โทริอิ
- **#8B6F47** (wood brown) ศาล + สะพาน
- **#6B8E5A** (sage green) ใบไม้
- **#F5F0E6** (cream) หิน

## Loading Strategy (zone-based)

Player เริ่มที่ Zone 2 (School - Hub)
- โหลด Zone ปัจจุบัน + 1 zone ข้างเคียง
- Stream zone อื่นเมื่อใกล้ (< 50m)
- Unload zone ที่ห่าง (> 150m)

**Total assets loaded at peak:** 3 zones = 60 buildings, 30 characters
**Memory budget:** < 500MB
